# F-Impose

Imposition tools for indie printmaking

Try it out: https://gfrancine.github.io/f-impose/

# Contributing

To suggest features or report issues, please [create an issue](https://github.com/gfrancine/f-impose/issues) in this repository or [message me personally](https://instagram.com/gracefrancines)!

# Developing

F-impose is built with [Vite](https://vite.dev/) and requires a NodeJS version of 20.19 or higher. See the [Vite documentation](https://vite.dev/guide/) for more details.

```sh
# Install dependencies
npm i

# Open a live development server
npm run dev 

# Build for production
npm run build

# Format the codebase, always run pre-commit
npm run fmt

# Run tests
npm run test

# Deploy to GitHub Pages
# Merges the master branch to the `github-pages` branch
# and pushes it upstream to the `github` remote.
npm run deploy-gh
npm run deploy-gl # equivalent command for Gitlab Pages
```
